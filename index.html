There are some unsupported languages in windows phone 7. And the platform is really nice, so its so unfair!! This program is first to deal with georgian language. With it you can type text in georgian and then send email/sms/do web search/just copy(paste) in georgian, so that in other programs you still cant see what are you sending, but thats just because of platform unsupporting georgian symbols, but it WILL still send them as georgian symbols! Also you can copy any text in "unsupported georgian" from other programs and read it in this one.

features

  • can write in georgian (using some sort of Sylfaen font)
  • can send emails, write sms, do web search and copy text(to paste in other locations)
  • can open copied georgian text from other programs
  • has two versions: with adds(no functional restrictions) and without them(for very small cost)
